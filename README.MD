# Обрезка ссылок с помощью вк 

Обрезаем ссылки и считаем сколько людей перешло по ней

## Как установить? 

python 3.10+ должен быть уже установлен на вашем компьютере. Переходим в терминал и устанавливаем зависимости:

- ```pip install -r requirements.txt```

Далее устанавливаем переменные окружения 

Зарегистрируйтесь в https://dev.vk.com/ru/reference и получите уникальный токен

- создаем рядом с main.py файл .env и создаем переменную окружения ```VK_TOKEN```
- ```VK_TOKEN=Ваш_токен```

### Запуск

1. Сокращение ссылок:
- ```python main.py 'ссылка' ( пример https://example.com/very/long/path )``` 

2. Получение статистики по ссылке:

- ```python main.py 'сокращеная ссылка' ```

- Примечание: ссылку желательно обернуть в ковычки во избежании ошибки
